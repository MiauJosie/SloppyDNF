# SloppyDNF

A "sloppy" but functional TUI frontend for the DNF package manager, built with [Textual](https://github.com/Textualize/textual).

## Features

- **Tabbed Browsing:** Easily switch between `all`, `installed`, and `updates` packages.
- **Instant Search:** Filter through the package list as you type.
- **Detailed Info:** View comprehensive package information, including descriptions, versions, and sizes.

## Prerequisites

Since this is a DNF frontend, you'll need to be on a system that actually uses DNF (Fedora, RHEL, CentOS, etc.).
